Bugs, Feature and Backport Requests¶

Bugs, features and backport requests for pulpcore are tracked with GitHub Issues. Please see the plugin table for trackers for each plugin. Use this link to file a bug or feature request.

Warning

Security bugs should be email to pulp-security@redhat.com with your Pulp version, vulnerability description, and reproduction steps.

Triage¶

Once per week, the Pulp team triages all new bugs and feature requests. See the meetings page on the website for more info on when it occurs and how to participate.
